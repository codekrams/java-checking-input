# java-checking-input
Das Programm soll den Nutzer aufffordern 1 oder 2 einzugeben (exemplarisch für ein Auswahlmenue). Bei Falscheingabe soll ein Fehlerhinweis ausgegeben werden und die Eingabe wiederholt werden, solange bis die Eingabe korrekt war. Die Anzahl der Fehler soll mitgezählt werden und am Enden ausgegeben werden
